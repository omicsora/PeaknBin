Publicly available ChIP-Seq data collection and analysis



The raw ChIP-Seq datasets of available proteins were collected from Gene expression omnibus (GEO) using a search query to specific to mouse embryonic stem cells (mESCs).

Search Query



mus musculus[Organism] AND "Genome binding/occupancy profiling by high throughput sequencing"[DataSet Type] AND ("embryonic stem"[All Fields] OR "ES"[All Fields] OR "mES"[All Fields] OR "mESC"[All Fields] OR "mESCs"[All Fields] OR "ESCs"[All Fields] OR "ESC"[All Fields])


The search query had given 1890 hits as on 01.12.2021 comprising 321 TFs.


Analysis Pipeline:

The raw fastq files were downloaded using SRA-Toolkit (NCBI). Low quality reads (phred < 20) and shorter reads were filtered out using Trimmomatic (v0.36) (Bolger et al., 2014). Since samples had different read lengths and different read modes (single and paired-end), we implemented a different pipeline accordingly. The alignment was performed to either mm9 or mm10 mouse reference genome with bowtie (v1.2.1.1) or with bowtie2 (v2.3.3.1). Uniquely mapped reads were obtained either by setting -m 1 option in bowtie1 or by applying samtools view -q 20 to bowtie2 aligned output file. The reads were deduplicated using a custom shell script. Peak calling was done using MACS1 (v1.4.2) (Zhang et al., 2008). In bedgraph files, scores were converted to reads per million (RPM) and converted to bigwig for uploading to UCSC genome browser (genome version mm9 or mm10).
